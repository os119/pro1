# Django Library Management Project

This repository includes my Django-based library management project.

I originally built this project during my early practice with Django to get more familiar with how web applications are structured. It mainly focuses on core concepts such as routing, models, forms, authentication, and CRUD operations.

Working on this helped me build a better understanding of how Django projects are organized, especially when dealing with user accounts, database models, and basic book management features.

## Main areas covered
- Django project structure
- User authentication
- CRUD operations
- Models, forms, and views
- Basic book and library management features

## Notes
This project reflects part of my early learning process in Django and web development, and I decided to keep it here as part of my GitHub portfolio.
